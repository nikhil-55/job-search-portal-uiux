This repository contains the UI/UX design work for a Job Search Portal created as part of an internship assignment.

## Project Overview
The goal of this project is to design a clean and user-friendly job portal that helps users easily search for jobs, view job details, and apply smoothly.

## Screens Designed
- Job Listing Page (Search & Filters)
- Job Details Page
- Application Flow & Confirmation

## Design Flow
1. User lands on the Job Listing page
2. Applies filters to refine job results
3. Selects a job to view details
4. Clicks Apply Now
5. Receives application success confirmation
